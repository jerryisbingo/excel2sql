# excel to sql
excel转sql语句，在线EXCEL转SQL insert语句的小项目。  
  
>springboot + redis + amazeui，其他辅助：sitemesh3+hutool  
 
 ---
   
[在线体验，暂未上传](http://e2s.usgame.com.cn/)  
   
项目clone下来后只需要改配置文件的redis即可。  
   
还有很多细节未加：    
  1.日志数据库log  
  2.文件大小配置，续传等问题  
  3.转换采用for循环，待优化  
  4.已转换的文件定时删除
